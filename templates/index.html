Recommendation Api Documentation

@movielens Dataset

Api Endpoints

/users

Gives a list of user-id's present in the movielens(100k) dataset.Movielens dataset holds user-id's in the range [1,943].

/users

/users/user_id

Gives the metadata information corresponding to user with id user-id.This includes information of age,occupation and sex about the user.e.g:check for user with user_id: 67

/users/67

/movies

Gives a list of movie-id's present in the movielens(100k) dataset.Movielens dataset holds user-id's in the range [1,1682].

/movies

/movies/movie_id

Gives the metadata information corresponding to the movie with id movie-id.This includes information of imdb_url,title,video release date, and 18 different genre descriptions based on binary confirmations,and unknown flag in case genre not known.e.g:check for user with movie_id: 342

/movies/342

/query/user_id

Gives a list of at-max 5 recommendation for the user with id user-id, along with the degree of recommendation..e.g.: recommendations for user with user-id:652

/query/652

/query/user_id/no_of_recommendations

Gives a list of no_of_recommendations for the user with id user-id,along with the degree of recommendation.e.g.:recommendations for user with user-id:87 and no. of recommendations required equal to 10.

/query/87/10

/query/user_id/k_nearest_neighbors/no_of_recommendations

Gives a list of no_of_recommendations for the user with id user-id,along with the degree of recommendation.In this case value of nearest neighbours k_nearest_neighbors specified in parameter is taken into account while producing recommendations. e.g.: recommendations for user with user-id:349, no. of k-nearest-neighbors equal to 20, and no. of recommendations required equal to 5. Note that the value of k-nearest-neighbors was default set to 5 whenever not specified, and the no. of recommendations required were default set to 5.

/query/349/20/5

About

  • Api uses recommenadtion system generation based on movielens-100k datset.Recommendation engine is build over user-based collaborative filtering using pearson correlation similarity measure computed with k-nearest neighbouring algorithm(k=10).
